# CV-In-HTML

A personal CV/Resume website built with HTML.

## Description

This project is a web-based CV showcasing personal information, education, work experience, skills, and languages.

## Files

- `index.html` - Main landing page
- `personaldata.html` - Personal information page
- `edudata.html` - Education details
- `workexperince.html` - Work experience
- `skills.html` - Skills and competencies
- `languages.html` - Language proficiencies

## How to Use

Simply open `index.html` in your web browser to view the CV website.

## Features

- Clean HTML structure
- Multiple sections for different CV components
- Image assets for visual enhancement
